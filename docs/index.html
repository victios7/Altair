Un lenguaje con servidor HTTP incorporado en la sintaxis.

Altair es un lenguaje compilado y expresivo: declaras rutas, jobs, sesiones y variables con nivel de almacenamiento propio, y altairc lo transpila a C y compila un binario nativo sin dependencias.

Instalación

De cero a compilando en tres pasos

El instalador añade altairc al PATH del sistema y deja un acceso directo de la terminal en el escritorio.

Descarga el instalador

Baja Altair-Setup.exe desde la sección de Releases en GitHub. No necesitas instalar nada más: el compilador incluye su propio toolchain (mingw64).

Ejecuta el instalador

Acepta los permisos de administrador (necesarios para escribir en el PATH del sistema) y sigue el asistente. Windows puede mostrar un aviso de SmartScreen por ser un instalador nuevo — pulsa Más información → Ejecutar de todas formas.

Abre Altair Terminal

Usa el acceso directo del escritorio, o abre una terminal normal y comprueba la instalación con altairc --version. Para compilar tu primer programa: altairc hola.at -o hola.

Características

Todo lo necesario para un servicio, en un solo archivo .at

01

Storage tiers

Cada variable declara dónde vive: ram, disk, cache o temp, con expiración y prioridad opcionales.

02

Servidor HTTP nativo

listen y route declaran endpoints directamente en el lenguaje, con middleware y rate limiting integrados.

03

Sesiones y config

Sesiones con TTL y variables de entorno tipadas mediante bloques session y config.

04

Jobs programados

Tareas recurrentes con job … every, ideales para limpieza, checkpoints o heartbeats.

05

Salud y métricas

Endpoints /health y /metrics compatibles con Prometheus con una sola línea.

06

Compila a binario nativo

Todo transpila a C y se compila con gcc: un único .exe sin runtime que instalar.

Ejemplo

Una API con auth, rate limiting y healthcheck

/ Configuración
config;
  env("PORT") default "8080"
  env("API_SECRET") required
break

health "/health";
  check "storage" -> 1
break

listen PORT;
  route "GET" "/todos" rate_limit 60 per_minute;
    respond.json(todos)
  break
break
