Qué cubre este playground: cabecera altair.doc, variables (numeric, text, bool, list), storage tiers ram/disk/cache/temp (simulados con localStorage del navegador), operadores, if/elif/else, while, repeat, forever, foreach, exit, funciones (fun/return, con recursión), try/catch, listas y log.

Qué NO cubre: class/object, token, snapshot, choose, y toda la capa de servidor (listen, route, config, health, sesiones, jobs, base de datos) — eso requiere el compilador real altairc + gcc corriendo en un servidor, algo que un navegador no puede hacer. Para eso, descarga Altair e instálalo localmente.
