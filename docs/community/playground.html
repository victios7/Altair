Prueba código Altair directamente aquí, sin instalar nada. Corre con un intérprete en JavaScript en tu navegador — no es el compilador oficial altairc, así que algunas funciones no están disponibles (ver nota abajo).

Pulsa "▶ Ejecutar" para ver el resultado aquí.
Qué cubre este playground: variables (numeric, text, bool, list), storage tiers ram/disk/cache/temp (simulados con localStorage del navegador), operadores, if/elif/else, while, repeat, forever, foreach, exit, funciones (fun/return, con recursión), try/catch, listas y log.

Qué NO cubre: class/object, token, snapshot, choose, y toda la capa de servidor (listen, route, sesiones, config, jobs, base de datos) — eso requiere el compilador real altairc + gcc corriendo en un servidor, algo que un navegador no puede hacer. Para eso, descarga Altair e instálalo localmente.
